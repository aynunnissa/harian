**STEPS:**

`git clone https://github.com/2ik/File-storage-laravel.git`

`composer install`

`copy .env.example .env`

`php artisan key:generate`

`php artisan migrate`

`php artisan db:seed`

`php artisan storage:link`

`php artisan serve`
